This tutuorial will tell you step by step on how to set up your personal Auburn webiste. The final result is (http://www.auburn.edu/~czj0043)

You need to know

Maybe many of students still do not know that:

The UNIX server, known as Mallard, hosts the Auburn University homepage. All Students and Employees automatically have 10GB of web space already set up on this server (http://www.auburn.edu/~username)

According to: AU OIT Web Development

jemdoc introduction

a. Install jemdoc

In fact, jemdoc is a python script. you can download it from (http://jemdoc.jaboc.net/dist/jemdoc.py), at the same time, download their default css file (http://jemdoc.jaboc.net/dist/jemdoc.css).

Basically, the grammer of jemdoc is very simialr to the markdown lanuage. You can build a very minimalist personal hompage using jemdoc with a your own customized css file.
